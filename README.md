# TypeShift

**[TypeShift](https://d7tq3ju7n5fb9.cloudfront.net/)** is a modern, minimalist web-based platform designed to help users practice, track, and improve their typing speed and accuracy. Inspired by high-performance typing tools, it provides a distraction-free environment with real-time analytics and global rankings.

## 🚀 Features

*   **Advanced Typing Engine** – Infinite typing support with a 4-line windowed scrolling system for a seamless experience.
*   **Real-time Analytics** – Live calculation of WPM (Words Per Minute) and accuracy during every test.
*   **Performance Dashboard** – Visualized progress tracking using dynamic line graphs (via Recharts) showing speed and accuracy trends.
*   **Global Leaderboard** – Compete for the top spot across multiple time-based modes (15s, 30s, 60s, 120s).
*   **Secure Authentication** – Full user accounts with JWT-based session management and encrypted data storage.
*   **Minimalist UI** – A sleek, dark-mode aesthetic designed for focus and clarity.

## 🛠️ Tech Stack

### Frontend
*   **React 19** – Modern UI components and state management.
*   **Vite** – Blazing fast build tool and dev server.
*   **Recharts** – Interactive data visualization for performance metrics.
*   **Lucide React** – Clean and consistent iconography.

### Backend
*   **Node.js & Express.js** – Robust RESTful API architecture.
*   **Drizzle ORM** – Type-safe, high-performance database interactions.
*   **JWT & Bcryptjs** – Secure authentication and password hashing.

### Database
*   **PostgreSQL** – Reliable relational data storage (running on **Amazon RDS** for production).

### Infrastructure (AWS)
*   **Amazon S3** – Static asset hosting for the frontend.
*   **Amazon CloudFront** – Global CDN with HTTPS and SPA routing support.
*   **Amazon EC2** – Reliable hosting for the containerized backend API.
*   **Docker** – Consistent environment orchestration.

## 📈 Project Status
